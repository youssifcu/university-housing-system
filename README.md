# University Housing System

**Description:** Full-stack university housing management system including website, admin dashboard, and mobile app built with React, React Native, Node.js, Firebase, and MongoDB.

**Branches:**
- main → stable code
- dev → main development branch
- feature-login, feature-register, feature-dashboard → feature branches

**Instructions for team:**
1. Checkout your branch
2. Pull from dev before you start
3. After finishing your work, commit + push to your branch
4. Open a Pull Request to dev
